# 🚀 Student Resource Sharing Platform

A full-stack MERN application that allows students to upload, manage, and access academic resources efficiently.

---

## 📌 Features

- 🔐 User Authentication (JWT-based login/signup)
- 📤 Upload Study Resources (PDF only)
- 🔍 Search Resources by Title
- 📚 Organized Resource Management
- 🧑‍💻 User Dashboard (View & Delete Uploads)
- ⚡ REST API Integration
- ☁️ MongoDB Atlas Database

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Atlas)

### Other Tools
- JWT Authentication
- Multer (File Upload)
- Git & GitHub
